### 조건문
- if문
- if-else문
- if-else if문
- switch문(break 없으면 다음 case 진행)

### 반복문
- for문
- while문
- do-while문
- break문 : break문이 수행되면 현 블럭의 반복문을 벗어난다.
- continue문 : continue문이 수행되면 블럭의 끝으로 이동한다. break문과는 달리 반복문을 벗어나지 않는다.
- 이름 붙은 반복문
```java
outer: while(true){
  for(;;){
    if(num==0)
      break; // for문을 벗어난다. while문 다시 실행.
    if(num==99)
      break outer; // 전체 종료. for문과 while문 모두 벗어난다.
  }
}
```
